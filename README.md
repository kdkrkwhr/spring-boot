## Install

#### 1. Open JDK Install

* JDK Install

```bash
$ yum install java-${version}-openjdk
$ yum install java-${version}-openjdk-devel
```

* 환경변수 등록

  * location =&gt; /usr/bin/java

  ```text
  //# vi /etc/profile

  ...

  JAVA_HOME=${location}
  PATH=$PATH:$JAVA_HOME/bin
  CLASSPATH=$JAVA_HOME/jre/lib:$JAVA_HOME/lib/tools.jar

  export JAVA_HOME PATH CLASSPATH
  ```
